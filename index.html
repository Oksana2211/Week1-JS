Вопросы:

  1. Какие бывают варианты объявления переменных?

    2. Ответ: Var (устаревший вариант), Const (неизменное значение), Let

  2. Как можно вывести сообщение "Hello World" с помощью JavaScript?

    3. Ответ: console.log('Hello, world!')

  3. Что выведет этот скрипт?

    4. Ответ:
    Hello, 1
    Hello, name
    Hello, Ilya

  4. Какое значение будет в z?

    5. Ответ: 7

  5. Какими тремя способами можно подключить JavaScript код?

    6. Ответ:

    • внутри тегов script/script
    • script src="путь_к_файлу"/script
    • используя специальные атрибуты — например, onclick, onmouseover, onkeypress, onload (они добавляют на элементы обработчики событий)


  6. Есть ли какая-то разница между записями typeof str и typeof(str)?

    7. Ответ: Он работает со скобками или без скобок, результат одинаковый

  7. Что вернет код?

    8. Ответ: 28

  8. Какой результат будет у выражений ниже?

    9. Ответ:

    "" + 1 + 0 = 10

    "" - 1 + 0 = -1

    true + false = 1

    6 / "3" = 2

    "2" * "3" = 6

    4 + 5 + "px" = 9px

    "$" + 4 + 5 = $45

    "4" – 2 = 2

    "4px" – 2 = NAN

    7 / 0 = Infinity

    " -9 " + 5 = -9 5

    " -9 " – 5 = -14

    null + 1 = 1

    undefined + 1 = NAN

    " \t \n" – 2 = -2

  9. Что будет в x?

    10. Ответ: 2

  10. Что будет в x?

    11. Ответ: 3
